# classification_project

## 1. Project Title
Breast Cancer Diagnosis using Scikit-learn (WDBC Classification)

## 2. Project Purpose
This project is designed to perform binary classification on the Breast Cancer dataset. It aims to train and compare multiple machine learning classifiers to determine which model provides the highest accuracy in predicting malignant or benign tumors.

## 3. Dataset Used
- **Source**: Built-in scikit-learn Breast Cancer dataset (Wisconsin Diagnostic Breast Cancer).
- **Details**: The dataset contains 569 samples with 30 features each, used for binary classification (malignant/benign).

## 4. How to Install Requirements
To run this project, ensure you have the necessary libraries installed. You can install them using the following command:

pip install -r requirements.txt


## 5. How to Run the Code
Execute the main script using Python:

python wdbc_classification.py

## Output Files
wdbc_classification_matrix.png: Confusion matrix visualization of the best-performing model.

wdbc_classification_scatter.png: Scatter plot visualization of the dataset features.

short_report.txt: A detailed report covering methodology, results, and analysis.
